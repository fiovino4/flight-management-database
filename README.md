# Flight Management Database (SQLite + Python CLI)

A beginner-friendly database-driven application for an airline that manages **flights, pilots, destinations, aircraft**, and **pilot assignments**.

This project demonstrates:
- **CRUD** operations (Create / Read / Update / Delete)
- **SQL JOINs** (e.g., pilot schedule, flight listings with destination codes)
- **Filtering** (destination / status / departure date)
- **Aggregation** reports using **GROUP BY**
- **Database constraints, indexes, and triggers** to enforce data integrity

---

## Files in this repository

- `schema.sql`  
  Creates all tables, primary keys, foreign keys, constraints, indexes, and triggers.

- `populate_db.py`  
  Rebuilds the database (`flight_management.db`) from scratch and inserts sample data.

- `flight_management.db`  
  The SQLite database file generated by `populate_db.py`.

- `main.py`  
  The command-line interface (CLI) used to interact with the database.
